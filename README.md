### Descripción:
Este proyecto consiste en la limpieza, normalización y migración de datos de una cafetería universitaria que presentaban inconsistencias y errores. A través de Python y la librería Pandas, se procesaron archivos CSV con datos sucios para convertirlos en información estructurada y confiable. Posteriormente, los datos fueron almacenados en una base de datos relacional SQLite, donde se implementó una arquitectura con llaves primarias y foráneas, además de operaciones CRUD para gestionar las ventas. El objetivo principal fue transformar datos desorganizados en un sistema eficiente y apto para análisis en herramientas como Power BI.
